# r60 std #

This is just a simple poker 60% board as a base for testing
stuff.

It's a 16MHz Atmega32U4 running [quantum mechanical
keyboard](https://github.com/qmk/qmk_firmware) firmware.

It should fit in most (all?) 60% poker cases.
